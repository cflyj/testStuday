# 滴滴实习期项目与技术总结

> 来源：`readme5.md` 提炼。聚焦平台建设、组件优化、数据可视化与性能提升。

## 概览
- 实习部门：滴滴用户增长/安全事务等
- 主要方向：数据平台、可视化、组件性能优化、业务中台、智能外呼系统等

---

## 主要项目与技术实践

### 1. W平台线索池与动态表单
- 负责线索池功能扩展，支持多业务线（如电商假货共建），通过动态表单解析 JSON 自动生成表单项，提升业务适配与扩展性。
- 组件长列表引入虚拟列表与 IntersectionObserver 懒加载，解决大数据量渲染卡顿。

### 2. P平台安全研判与图谱分析
- 参与安全研判平台力导向聚类图、单点血缘图等复杂可视化开发，支持多层级用户关系分析与交互。
- 自研环形布局算法，支持节点溢出分段排列与窗口展示，提升大规模团伙分析效率。

### 3. 大盘组合分析模板与后台配置
- 联合数仓团队共建大盘组合分析模板，实现多图表联动、灵活配置与核心指标下钻，提升业务看数效率。
- 封装 useDrawer、useImperativeHandle 等自定义 hooks，优化后台配置体验与组件复用性。

### 4. 数据可视化与性能优化
- 针对 ECharts 大数据渲染卡顿，落地 dataZoom 分段加载、sampling 降采样、largeThreshold 高性能模式等多种优化方案。
- 前后端协作聚合数据，结合前端虚拟列表，显著提升大屏与看板的流畅度。

### 5. 其他工程实践
- IvR 智能外呼系统兜底逻辑开发，D3 拖拽与矢量卡片实现。
- soya 管理系统前端负责人，落地 JWT 持久化登录、乐观更新、axios 二次封装、虚拟列表等。
